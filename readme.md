
# Django Blog Project

Django-Blog is blogging CMS. It's using Django-web-framework as backend and Django-REST-framework for it's API.

## Here is some of it's features

- User management system
  - List of user posts
- Dashboard
- Post
  - Picture
  - Read more
  - Thumbnail
- Category
- Comment system
- Stats
- Captcha system
